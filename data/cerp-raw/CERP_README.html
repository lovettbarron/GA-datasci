Commander's Emergency Response Program Data -- Raw FOIA file

Released January 2015

PLEASE NOTE: This is raw data released to ProPublica under a Freedom of Information Act. The data was culled together from several different databases by the Special Inspector General for Afghanistan Reconstruction (SIGAR). (You can read more about the databases here.) This means there are, in many cases, both duplicate records and duplicate fields. The best way to deal with the duplicate records is to filter the field "Same Report" to 0 (zero). The field "Report Key Repeats" shows how many times the record is repeated in the database.

Also, this database was entered by field personnel, and contains some estimates, errors and misspellings. Report with caution when using this data. For more information on the contents of this database, contact SIGAR here.
